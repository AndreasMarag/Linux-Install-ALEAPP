# Linux-Install-ALEAPP

How to install ALEAPP in Linux (UBUNTU 22.04 LTS 64-bit)

1 Go to the project's repo "https://github.com/abrignoni/ALEAPP", read the Requirements, and install the Dependencies.

2 Open the terminal and clone the project "git clone https://github.com/abrignoni/ALEAPP.git".

3 Cd to the folder you cloned the repository and run the command "python3 aleappGUI.py".

4 Enjoy :)))))))))
